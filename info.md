# Neatsvor Integration

Управление пылесосами Neatsvor и Black Vision в Home Assistant.

## Скриншоты



## Особенности

- ✅ **Зональная уборка** с карты
- ✅ **Живая карта** с позицией робота
- ✅ **Уборка комнат** по названию комнаты
- ✅ **Русская локализация**
- ✅ **История уборок** с картами
- ✅ **Облачные карты** для восстановления
- ✅ **YandexSmartHome** поддержка голосового контроля и управления

## Требования

- Home Assistant 2024.1+
- HACS (для простой установки)

## Быстрый старт

1. Установите через HACS
2. Добавьте интеграцию через UI
3. Настройте карточку через `xiaomi-vacuum-map-card`
